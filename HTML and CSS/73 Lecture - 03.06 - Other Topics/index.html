Libraries and Frameworks

Library - someone's else written code you can use in your own project.
To use a library you simply need to import it's CDN to your project and use it's classes.

CSS Libraries

  • Bootstrap
  • Tailwind
  • Bulma
  • UI Kit
  • Materialize
  • Material Design Lite
  • Milligram
  • ...

Pros

  • Makes styling HTML code way faster
  • Reduce the need of writing CSS code
  • Integrated responsiveness
  • Helps to reduce redundant code while working in teams
  • ...

Cons

  • You forget how to write regular CSS
  • Libraries can not do everything for you, you still need to know basic CSS
  • Polute HTML elements with bunch of classes instead of using different selectors
  • Can get into a problem while trying to apply your own style due to specificities of selectors
  • Have to learn the library before it becomes useful
  • Not unique design of websites using same library
  • Using libraries makes you use bunch of bundled code, from which you won't even use 10% of classes
  • ...
