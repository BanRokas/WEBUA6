Box Model

  1. Content - center part of element
  2. Padding - in between content and border
  3. Border - in between padding and margin
  4. Margin - in between our element and surrounding elements
  5. Position - read more in next section
Box Model example

Margin

Inline (display) elements can't have margin to top and bottom.
Inline and Inline-Block (display) elements can't have margin set to auto. Because they do not have any reserved space.

Block, Grid and Flex (display) elements can have "margin: anySize auto;" to center an element.

Positions

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky
  • top, right, bottom, left, z-index (not positions)

Thenever we give position (not static) to an element, we get to use: top, right, bottom, left, z-index styles.
Top, right, bottom and left works differently with different positions.
z-index always works the same - it's used to set element's layer.

Fonts from google fonts

  3. Add your desired Font by clicking "Get font" button at top right of page

Icons

To use icons you need to import them to your project using CDN links to HTML or CSS.
To display icons you need to use either "i" or "span" tags with specific class names.
If you want to style the icons, think of them as text elements.
