Flex

We use display of Flex then we want to display our elements on the screen with a bit more control over them. Even though we do not have too much control with every single element separately, we do control them as a group quite nicely and easily.
Flex got quite a few additional properties for parent element (flex container) and children elements (flex items).

Flex properties

Parent

  • display: flex;
  • flex-direction - in which direction elements are displayed
  • flex-wrap - do elements wrap to the other line or not
  • flex-flow: flex-direction and flex-wrap;
  • justify-content: how elements displayed on main axis
  • align-content: how elements displayed on cross axis
  • align-items: how elements displayed on cross axis
  • row-gap
  • column-gap
  • gap - gap in between children elements (row and column)

Children

  • order - you can swap the order of children elements
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex: flex-grow flex-shrink flex-basis;
  • align-self - align-items properties for individual item
