Flex

We use display of Flex then we want to display our elements on the screen with a bit more control over them. Even though we do not have too much control with every single element separately, we do control them as a group quite nicely and easily.
Flex got quite a few additional properties for parent element (flex container) and children elements (flex items).

Flex properties

Parent

  • display: flex;
  • flex-direction - in which direction elements are displayed
  • flex-wrap - do elements wrap to the other line or not
  • flex-flow: flex-direction and flex-wrap;
  • justify-content: how elements displayed on main axis
  • align-content: how elements displayed on cross axis
  • align-items: how elements displayed on cross axis
  • row-gap
  • column-gap
  • gap - gap in between children elements (row and column)

Children

  • order - you can swap the order of children elements
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex: flex-grow flex-shrink flex-basis;
  • align-self - align-items properties for individual item

Grid

We use display of Grid, then we want to display elements inside of that grid parent element with all the control over the children element we can want. But we do need to create the grid itself, by telling how many and what sized columns and rows we create inside of the grid AND then we need to tell each children what position inside of that grid they take.
Grid got quite a few additional properties for parent element (grid container) and children elements (grid items).

Grid Properties

Parent

  • display: grid;
  • grid-template-columns
  • grid-template-rows
  • grid-template: grid-template-rows / grid-template-columns
  • grid-template-areas
  • row-gap
  • column-gap
  • gap: row-gap column-gap;
  • align-content
  • justify-content
  • place-content: align-content justify-content;
  • align-items
  • justify-items
  • place-items: align-items justify-items;
  • grid-auto-columns
  • grid-auto-rows
  • grid-auto-flow
  • grid: grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns, grid-auto-flow;

Children

  • grid-row-start
  • grid-column-start
  • grid-row-end
  • grid-column-end
  • grid-row: start / end
  • grid-column: start / end
  • grid-area: row-start / column-start / row-end / column-end;
  • align-self
  • justify-self
  • place-self

Units and Functions

  • fr
  • min-content
  • max-content
  • auto
  • fit-content()
  • minmax(min, max)
  • repeat(howMany, ofWhat)
  • masonry
  • subgrid
