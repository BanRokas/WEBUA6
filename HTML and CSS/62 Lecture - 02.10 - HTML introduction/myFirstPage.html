Tag list

  • h1-h6 - heading - displaying headings
  • p - paragraph - displaying paragraphs
  • img - image - displaying images
  • div - - grouping tags in blocks
  • span - - isolating some of text content
  • ul - unordered list - creates unordered list
  • ol - ordered list - creates ordered list
  • li - list item - element in the list
  • button - - creates a button like element
  • a - anchor - creates a link
  • br - line break - breaks line
  • hr - horizontal line - breaks line and creates a horizontal line
  • table - - creates a table group
  • thead - table head - group for column names
  • tbody - table body - group for table data
  • tfoot - table foot - group for some kind of summaries
  • tr - table row - creates a row
  • th - table heading - name of the table column
  • td - table data - table data

Attribute list

  • src - source - path to file
  • href - hyperlink reference - URL/link
  • alt - alternative text - displayed in case of wrong path
  • width - - sets width of media element
  • height - - sets height of media element
  • target - - controls where new link is being opened
  • rowspan - row span - write a number of how many rows that element should take
  • colspan - column span - write a number of how many columns that element should take

  • CTRL A - select all

  • CTRL Y / CTRL SHIFT Z - cancelling cancellation
