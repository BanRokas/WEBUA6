Semantic tags

  • header - top part of the page (menu, login, sign in...)
  • main - main content of the page
  • footer - displaying: copyrights, terms and uses, cookies, privacy policy...
  • nav - navigation - used to help user to navigate around the website
  • section - used to display a big group of elements about the same topic
  • article - used to display a big group of elements about the same topic
  • div - used to group elements together

Cascading Style Sheets

Ways to write CSS

  • Inline - write CSS in tag's style attribute
  • Internal - to write CSS in HTML file, in style tag (which is normally placed in a head)
  • External - to write CSS in a separate file and link it to the HTML

Inline

Inline CSS is normally used to test out styles or to inject a specific style onto HTML element from JavaScript.
In order to use inline style, we need to add attribute called "style" to the tag and write CSS code as it's value.

Internal

Internal CSS is normally used to test out styles or/and selectors inside of an HTML element without the need of creating additional CSS file.
In order to use internal style, we need to create "style" tag (normally inside of a head tag) and write CSS code in there.

External

External CSS is used most commonly (practically always).
In order to use external style, we need to create additional file of CSS type and link it to the HTML file inside of which we want the styles of CSS file to work.

Selectors

Selectors are part of CSS code which selects which HTML elements should have the styles that are written inside of those selectors.

  • tag - selects all the tags which names match the one in CSS code
  • multiple - using comma, we can write multiple selectors and give them all same styles
  • * - all selector
  • .class - make up some kind of name
  • #id - make up some kind of name

Class

To create a class in CSS file, we need to write "." and after it without any spaces we need to write the name of the class (name should be logical, short and describe what class is used for): ".nameOfTheClass".
To give HTML element any class, we need to give HTML element attribute of "class" and value of "nameOfTheClass".
We can give as many classes to a single element as we want AND we can use same class on multiple elements. (many-to-many relation)

Id

To create an ID in CSS file, we need to write "#" and after it without any spaces we need to write the name of the ID (name should be logical, short and describe what ID is used for): "#nameOfTheID".
To give HTML element any ID, we need to give HTML element attribute of "id" and value of "nameOfTheID".
We can give a single id to a single element AND we can use a single id only once. (one-to-one relation)
