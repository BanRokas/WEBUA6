Measurement units

heading 3 el

Measurement units are used to set elements sizes for: width, height, gaps, font-size, radiuses and so on...

  • px
  • % - % of parent element
  • vh - viewport height - visible webpage height (100 - full height)
  • vw - viewport width - visible webpage width (100 - full width)
  • fr - fraction - grid topic
  • rem - root em(ephemeral) - 16px - root setting (:root)
  • em - ephemeral - 16px - closest ancestor element size settings

Box-Sizing

Box sizing has only 2 settings: border-box and content-box (default).
Having border-box setting on - our elements do not enlarge from paddings or borders.

Multimedia elements

There are many different multimedia elements: pictures, music, the sound, video recordings, voice recordings, movies, animation etc...

The most interesting one is iframe tag. Using it you can embed parts other websites to your own website.

Selectors

Attribute selectors

  • [attribute] - attribute name
  • [attribute="value"] - attribute name with specific value
  • [attribute~="value"] - attribute name with specific word in the value
  • [attribute*="value"] - attribute name with part of word
  • [attribute^="value"] - attribute name that starts with value
  • [attribute&="value"] - attribute name that ends with value
  • [attribute|="value"] - attribute name with value or "-" and value

Pseudo Class selectors

  • Links
    1. :link - unvisited link
    2. :visited - visited link
    3. :hover - mouse over the element (not only for links)
    4. :active - reacts to the click (moment of clicking)
  • :first-child - element which is the first child
  • :last-child - element which is the last child
  • :only-child - element which is the only child in the group
  • :first-of-type - element which is the first child of it's type
  • :last-of-type - element which is the last child of it's type
  • :only-of-type - element which is the only child of it's type in the group
  • nth-child(n) - selects elements by the sequents
  • nth-last-child(n) - selects elements by the sequents from the end
  • nth-of-type(n)
  • nth-last-of-type(n)
  • No need to know
    • :root
    • :not(selector) - selects the ones who are not "selector"
    • :empty
    • :fullscreen
    • :lang(language)
    • :target
